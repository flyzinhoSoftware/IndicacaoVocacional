# Indicação Vocacional

Sistema de Indicação Vocacional desenvolvido em PHP usando arquitetura MVC.

## Estrutura do Projeto

- `app/`: Código da aplicação
  - `Controllers/`: Controladores
  - `Models/`: Modelos
  - `Views/`: Visualizações
  - `core/`: Classes base
- `config/`: Configurações
- `public/`: Ponto de entrada público
- `routes/`: Definições de rotas

## Instalação

1. Clone o repositório
2. Configure o banco de dados em `config/database.php`
3. Acesse via `http://localhost/projectos/IndicacaoVocacional/public/`

## Uso

Desenvolva controladores em `app/Controllers/`, modelos em `app/Models/` e views em `app/Views/`.
